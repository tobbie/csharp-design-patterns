
## Design Patterns in C# - A collection of the Gang Of Four patterns (WIP)
The project is a group of class libraries that contain creational, behavioral and structural design patterns 

### Creational patterns 
	* Prototype
	* Singleton
	* Factory Method
	* Abstract Factory
	* Builder

### Structural patterns 
	* Adapter
	* Bridge
	* Composite
	* Decorator
	* Facade
	* Flyweight
	* Proxy

### Behavioural patterns

	* Strategy
	* Command
	* State
	* Mediator
	* Chain of Responsibility
	* Template Method
	* Visitor
	* Iterator
	* Observer
	* Memento
